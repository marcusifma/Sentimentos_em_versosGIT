Sentimentos em Versos
Sentimentos em Versos é um projeto que celebra a literatura de cordel como forma de inclusão, educação e cultura. Criamos kits de cordéis, pensados para dois públicos: infantil e geral, oferecendo experiências que unem tradição e acessibilidade.

Sobre o Projeto
O projeto busca resgatar a magia dos cordéis, trazendo temas educativos e culturais em formatos adaptados para diferentes idades. Todos os cordéis são feitos com:

Letra grande para facilitar a leitura.

Versão em braille para acessibilidade.

QR codes que levam a áudio descrições, ampliando o acesso.

Público-alvo
Infantil: Cordéis com linguagem simples, ilustrações e atividades didáticas.

Geral: Cordéis com linguagem mais elaborada e temáticas diversas.

Principais Produtos
Kits com 4 cordéis (divididos entre infantil e geral).

Website oficial, contendo:

Catálogo de produtos.

Loja online.

Vídeos interativos.

Jogos educativos sobre cordel.

Missão
Promover a inclusão cultural através da literatura popular brasileira, tornando a arte do cordel acessível para todos os públicos.

Tecnologias e Ferramentas
HTML, CSS e JavaScript (para o site)

Recursos de acessibilidade (braille, audiodescrição)
